The basics of linked lists

By Elze Hamilton / @elze

Women Who Code technical interview preparation series

Linked lists are very minimalistic data structures

A linked list is a set of nodes, each of which has a reference to the next node.

All we "know" about a linked list is its head node.

Here is a C# or Java class that defines a node that contains an integer value. 


public class ListNode {
       int value;
       ListNode nextNode;
}

Here is a Javascript constructor function that constructs a ListNode object.


function ListNode(val, nextN) {
	this.value = val;
	this.nextNode = nextN;
}

Write a function that takes an argument n, and returns a head node of a list with n ListNodes that contain numbers from 1 to n. Here it is in Javascript.


function ListNode(val, nextN) {
	this.value = val;
	this.nextNode = nextN;
}
function createLinkedList(n) {
  if (n < 1)
    return null;
  var lNode = new ListNode(n, null);
  for (var i = n-1; i >=1; i--) { 
    var node = new ListNode(i, lNode);
    lNode = node;
  }
  return lNode;
}

Exercise 2. Solution.

Write a function that takes the head node as an argument, and writes out the values of all the nodes in the list.


public void writeOutAllNodes(ListNode headNode) {
	ListNode node = headNode;
	while (node != null) {
		Console.Writeln("node.value = " + node.value);
		node = node.nextNode;
	}
}
